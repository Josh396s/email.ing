#### Urgency

* **Low** - No action needed, no time sensitivity, read at leisure
* **Medium** - Action or awareness needed, but no immediate deadline
* **High** - Time-sensitive, requires action within 24-48 hours
* **Critical** - Immediate action required. Deadlines today, security alerts, emergencies.
